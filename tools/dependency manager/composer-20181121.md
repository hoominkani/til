# composerの使い方

## composerとは
 - パッケージ・ライブラリの依存関係を管理するツール

## 使い方

### 初めて環境を作る時
 - composer isntall

### パッケージを追加する時
 - composer require ${PACKAGE_NAME}:${VERSION}
  - 追加したパッケージがpackage.json、package.lockに自動で追記される。

### 新しいバージョンのパッケージが欲しい時
  - composer update

### パッケージを検索する時
 - composer search ${PACKAGE_NAME}
